## Setup Environment for development

Create a python virtual enviroment
```
pipenv install
pipenv run dev
```

## Features
- List vaults
- Create Vault
- Remove Vault
- Upload File to Vault
- List Archives of a Vault
- Start an inventory job to request the list of archives of a Vault
- Download an archive
- Delete Archive
